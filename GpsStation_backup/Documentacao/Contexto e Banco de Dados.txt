a pasta Contexto contém os contextos que fazem a manipulação do banco de dados

Usamos Entity Framework para gerenciar o banco de dados dinamicamente por meio de contextos

Entity Framework realiza as operações de BD pelo código do sistema

para utilizar o Entity Framework é necessário instalar os pacotes
microsoft.entityframeworkcore
microsoft.entityframeworkcore.design
microsoft.entityframeworkcore.tools
microsoft.entityframeworkcore.sqlserver (usaremos o sql server como SGBD), para outros bancos, instalar outras lib

configurar serviço de conexão no Program.cs

a string de conexão fica em appsettings.jason

após configurar o Program.cs, é necessário criar o Migration para criar a base de dados e a tabela de fato

pegará o conteúdo da Model desejada, que já está setando o contexto e criará um mapeamento para o banco de dados

tools/NuGet package manager/Pakege Manager Console

Add-Migration Inicial -Context Contexto (para a classe Contexto)
este comando cria apenas a Migration, depois é necessário executar o Update

Update-database -Context Contexto
este comando cria o banco e a tabela conforme a a classe Contexto
